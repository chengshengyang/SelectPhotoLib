# SelectPhotoLib
Select photo from local file system for Android

使用了UIL加载图片

使用了PhotoViewLib显示图片大图

详情参考libs文件夹

支持相册列表-ListView，图片列表-GridView，大图浏览-ViewPager

支持单选和多选
